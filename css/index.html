Learn to code CSS for begginers

CSS describes how HTML elements should be displayed. CSS is a language that describes the style of an HTML document.

CSS Colors

Colors in CSS are most often specified by:

  • a valid color name
  • a RGB value
  • a HEX value

CSS Backgrounds

The CSS background properties are used to define the background effects for elements.

CSS background properties:
  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

CSS Borders

The CSS border properties allow you to specify the style, width, and color of an element's border.
